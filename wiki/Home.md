# AuctionHouse Wiki

AuctionHouse is an ASP.NET Core MVC application for running online auctions with bid validation, user roles, dashboard insights, and admin moderation.

## Stack

- .NET 8
- ASP.NET Core MVC
- ASP.NET Core Identity
- Entity Framework Core 8
- SQLite (default)
